# Wedding Bot

Telegram бот для управления свадебным мероприятием с Mini App для регистрации гостей.

## Основные возможности

- Регистрация гостей через Mini App
- Управление списком гостей через Google Sheets
- Админ-панель для управления приглашениями
- Интеграция с Telegram группой

## Деплой

Проект использует Docker для деплоя на Render.com. См. `DOCKER_DEPLOY.md` для инструкций.

## Требования

- Python 3.11+
- Telegram Bot Token
- Google Sheets API credentials

## Переменные окружения

См. `DOCKER_DEPLOY.md` для списка всех необходимых переменных окружения.
